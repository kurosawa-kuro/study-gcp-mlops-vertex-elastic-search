# 実装済み一覧

---

## ファイル一覧

### ルート

| ファイル | 説明 |
|---------|------|
| `.env` | 設定値・認証情報（git管理外） |
| `.gitignore` | 除外設定（Terraform state, .env, __pycache__等） |
| `CLAUDE.md` | Claude Code向けプロジェクトガイド |
| `Makefile` | ビルド・デプロイコマンド統合エントリポイント |
| `README.md` | プロジェクト概要 |

### .github/workflows/

| ファイル | 説明 |
|---------|------|
| `terraform.yml` | Terraform CI/CD（PR→plan表示、main→apply自動実行） |

### src/（アプリケーション）

| ファイル | 説明 |
|---------|------|
| `main.py` | Cloud Run Job（ingest: データ投入 / search: キーワード検索） |
| `data/sample.json` | 投入用サンプルデータ（3件: Elasticsearch入門, Terraform入門, Cloud Run概要） |
| `Dockerfile` | Python 3.11-slim コンテナ定義 |
| `requirements.txt` | 依存（elasticsearch, google-cloud-secret-manager） |
| `.dockerignore` | ビルド除外設定 |

### terraform/（IaC）

| ファイル | 説明 |
|---------|------|
| `main.tf` | 全リソース定義（ec_deployment, Artifact Registry, Secret Manager, IAM, Cloud Run Job） |
| `providers.tf` | プロバイダ設定（ec ~> 0.12, google ~> 5.0） |
| `variables.tf` | 変数定義（project_id, region, deployment_name, job_name, repo_name, secret_name, elastic_cloud_api_key） |
| `outputs.tf` | 出力値（elasticsearch_endpoint, secret_name, cloud_run_job_name, artifact_registry_url） |

### scripts/（オペレーション）

| ファイル | 説明 |
|---------|------|
| `config.py` | 共通設定（.envからの環境変数読み取り、run, dispatch ユーティリティ） |
| `docker_ops.py` | Docker操作（build, push, docker-run, clean） |
| `gcp_ops.py` | GCP操作（ingest, search, logs） |
| `tf_ops.py` | Terraform操作（init, plan, apply, apply-infra, destroy, import） |

### docs/（ドキュメント）

| ファイル | 説明 |
|---------|------|
| `elastic-search.md` | 技術ガイド（アーキテクチャ、コード解説、Terraform設計） |
| `運用.md` | 運用ガイド（deploy-all/destroy-all、日常オペレーション、トラブルシューティング） |
| `仕様、設計書.md` | 仕様・設計書（リソース一覧、Terraform設計、アプリケーション設計） |
| `実装済み一覧.md` | 本ファイル |
| `log-sample/` | 実行ログサンプル |
